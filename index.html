CSS

  • A programming language used to define how webpage content is presented to users
  • Defines the color, aesthetics, visual layout, and organization.
  • Define style rules for specific elements of an HTML page and format them using a wide range of style options.
  • A margin indicates how much space we want around the outside of an element.

Git

  • A distributed version control system that is used to track changes in source code during the development process.
  • It allows multiple developers to collaborate on a project.
  • It track developers changes made by each other.
  • Give the ability to revert to previous versions if needed.
  • Enables developers to work on different features or fixes simultaneously without interfering with each others work.

Javascript

  • Is a high-level dynamic programming language that allows users to interact with website.
  • You can power your websites with effects, interactivity, data communication, and much more.
  • Is the tool that takes a static HTML/CSS webpage and makes it interactive.
  • Clicking on a button, retrieve data, or fill out a form that is JS at work.
